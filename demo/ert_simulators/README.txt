KHEPRA DEMO SIMULATORS - EXECUTIVE GUIDE
========================================

Use these scripts during Executive Roundtables (ERTs) to simulate the "Weapons Systems" analyzing their environment in real-time.
PRE-REQ: Python 3.x

HOW TO RUN
----------
Open your terminal to this folder:
cd "demo/ert_simulators"

1. STRATEGIC WEAPONS SYSTEM (Package A)
   Run: python package_a_readiness.py
   
   WHAT IT DOES:
   - Simulates parsing their "Confidential Strategy PDF".
   - Finds a "Strategic vs. Technical" mismatch (The "Oppenheimer" moment).
   
2. OPERATIONAL WEAPONS SYSTEM (Package B)
   Run: python package_b_architect.py
   
   WHAT IT DOES:
   - "Builds" a Digital Twin graph in the terminal.
   - Hunts for "Toxic Trust" in the supply chain (e.g. Log4j, SolarWinds).
   
3. TACTICAL WEAPONS SYSTEM (Package C)
   Run: python package_c_crypto.py
   
   WHAT IT DOES:
   - Hashes their "codebase" (matrix effect).
   - "Breaks" RSA/ECDSA keys on screen.
   - Migrates them to Kyber/Dilithium.

4. THE "GODFATHER" REPORT (Package D)
   Run: python package_d_godfather.py
   
   WHAT IT DOES:
   - The closer. Synthesizes all simulated findings into a Board-Level "Kill Switch" or "Save" decision.

5. LIVING TRUST CONSTELLATION (The Visualizer)
   Run: python package_e_dag_viz.py
   
   WHAT IT DOES:
   - Generates a local HTML file ('dag_viz.html') and opens it in your browser.
   - Uses D3.js to render a beautiful, growing DAG of nodes.
   - This represents the Khepra Network effectively "healing" or "securing" itself.
