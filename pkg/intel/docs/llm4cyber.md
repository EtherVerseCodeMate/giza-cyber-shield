# Knowledge Base: LLM4Cybersecurity

This document contains a curated list of research papers and resources regarding the application of Large Language Models (LLMs) in Cybersecurity. It serves as a foundational knowledge base for the Khepra Agent's RAG (Retrieval-Augmented Generation) capabilities.

## Introduction
Use this context when answering questions about:
- Building cyber-security oriented LLMs
- Benchmarking LLMs for security
- Applications in Threat Intelligence, Fuzzing, Vuln Detection, and Code Repair.

## Key Research Papers & Benchmarks

### Evaluation Benchmarks
- **PACEbench (2025)**: Evaluating Practical AI Cyber-Exploitation Capabilities.
- **AICrypto (2025)**: Benchmarking Cryptography Capabilities of LLMs.
- **CyberGym (2025)**: Evaluating AI Agents with Real-World Vulnerabilities.
- **Purple Llama CyberSecEval (2023)**: Secure Coding Benchmark.
- **SecurityEval (2022)**: Vulnerability examples for code generation.

### Threat Intelligence
- **DroidTTP**: Mapping Android Apps to TTPs.
- **LRCTI**: LLM framework for Evidence Retrieval/Reasoning in CTI.
- **Crimson**: Strategic Reasoning in Cybersecurity.
- **HuntGPT**: Integrating Anomaly Detection with Explainable AI.

### Vulnerability Detection
- **SEC-bench**: Automated Benchmarking of LLM Agents.
- **VulStamp**: Vulnerability Assessment using LLM.
- **Detect Llama**: Finding Smart Contract Vulnerabilities.
- **ChatGPT for Static Application Security Testing (SAST)**.

### Automated Program Repair (APR)
- **VulnRepairEval**: Exploit-based evaluation for APR.
- **RepairLLaMA**: Efficient representations for program repair.
- **InferFix**: End-to-End Program Repair.

### Offensive Security (Red Teaming)
- **PentestGPT**: LLM-empowered Automatic Penetration Testing.
- **AutoAttacker**: LLM Guided System for Automatic Cyber-attacks.
- **RatGPT**: Turning online LLMs into proxies for malware attacks.
- **Jailbreaking Generative AI**: Phishing attacks by novices.

### Anomaly Detection & Defense
- **LogGPT**: Log Anomaly Detection via GPT.
- **Audit-LLM**: Multi-Agent Collaboration for Log-based Insider Threat Detection.
- **Hacking Back the AI-Hacker**: Prompt Injection as Defense.

## Trends
1.  **Agentic AI**: Shift from static analysis to autonomous agents (Agent4Cybersecurity).
2.  **Specialized Models**: Fine-tuning Llama/Mistral/DeepSeek for specific security domains.
3.  **Hybrid Approaches**: Combining LLMs with static analysis (SAST) and fuzzing.
