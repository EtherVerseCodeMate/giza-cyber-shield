4. TECHNICAL DATA RIGHTS
================================================================================

4.1 **LIMITED RIGHTS IN TECHNICAL DATA**

The Government acquires "Limited Rights" in technical data related to the Software, as defined in DFARS 252.227-7013.

**Limited Rights** means the Government may:
- Use technical data within the Government
- Reproduce technical data for Government use
- Disclose technical data to other Government agencies

**Limited Rights** does NOT permit:
- Release of technical data to the public
- Use of technical data for commercial purposes
- Disclosure to foreign governments without Licensor consent
- Reverse engineering based on technical data

4.2 **PROPRIETARY TECHNICAL DATA**

The following technical data is PROPRIETARY and NOT provided to the Government:
- Source code for AdinKhepra-PQC Lattice algorithms

- Trade secret methodologies

- Licensor will mark GFI with appropriate distribution statements
- Licensor will not use GFI for commercial purposes without authorization
